# NEON — Pro Player · Leviatán VCT

Este es el proyecto oficial de la landing page de **Bruno "Neon" Rodríguez**, jugador profesional de Valorant para **Leviatán VCT Americas**.

El proyecto ha sido modernizado y estructurado usando **Vite** para ofrecer un rendimiento óptimo y una fácil integración con plataformas de despliegue como **Vercel**.

## 🚀 Tecnologías

- **Vite**: Framework de frontend ultra-rápido.
- **HTML5 & CSS3**: Diseño premium con glassmorphism, animaciones y scroll reveal.
- **Vanilla JavaScript**: Lógica ligera para interactividad y efectos.

## 🛠️ Instalación y Uso Local

Para ejecutar este proyecto en tu máquina local:

1. **Clona o descarga** los archivos del proyecto.
2. Abre una terminal en la carpeta raíz.
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre tu navegador en `http://localhost:5173`.

## 📦 Compilación para Producción

Si deseas generar los archivos optimizados para subir a un servidor:

```bash
npm run build
```

Los archivos finales se generarán en la carpeta `dist/`.

## 🌐 Despliegue en Vercel

Este proyecto está optimizado para **Vercel**:

1. Sube el código a un repositorio (GitHub/GitLab/Bitbucket).
2. En Vercel, selecciona **"New Project"**.
3. Importa tu repositorio.
4. Vercel detectará automáticamente que es un proyecto de Vite. Haz clic en **"Deploy"**.

## ✨ Características

- **Diseño Responsivo**: Adaptado para móviles y tablets.
- **Scroll Reveal**: Animaciones suaves al navegar por las secciones.
- **Configuración Gamer**: Sección detallada con settings de mouse, crosshair (con función de copiado) y hardware (Gear).
- **Logros**: Historial detallado de la carrera competitiva de Neon.
